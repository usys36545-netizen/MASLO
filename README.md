# Приложение "Масло масленное"

## 1. Описание приложения

Приложение 'Масло масленное' было разработано для познания основ готовки основных продуктов. Это кулинарный помощник с графическим интерфейсом, созданный с использованием библиотеки PyQt5, предназначенный для новичков на кухне или тех, кто хочет освежить свои кулинарные навыки. Приложение предоставляет понятные инструкции, полезные советы и пошаговые рецепты различных блюд. Интерфейс включает выпадающий список с различными категориями блюд (например, Пельмени, Рис, Макароны, Греча, Яички и др.) и возможность переключения между светлой и тёмной темами оформления.

## 2. Назначение файлов

В проекте используются следующие ключевые файлы:

- **MASLO.ui** - файл графического интерфейса, созданный в Qt Designer. Содержит визуальное описание элементов управления приложения (окна, кнопки, списки и т.д.). Именно в этом файле формируется внешний вид приложения.

- **MASSSSLOO.py** - основной Python-скрипт приложения. Содержит логику работы программы, обработчики событий, функции для загрузки интерфейса из .ui файла и реализацию всех функций приложения.

## 3. Как запустить у себя

 1 Скачать по ссылке инсталер и запустить установку https://drive.google.com/uc?export=download&id=16nUVoEY5Wiss1IWGF1XQMbCbZkYwXBdk

 2 Для запуска приложения на вашем компьютере выполните следующие шаги:

   1. Убедитесь, что у вас установлен Python
   2. Установите библиотеку PyQt5:
      ```bash
      pip install PyQt5
      ```
   3. Скачайте файлы проекта (MASLO.ui и MASSSSLOO.py) на ваш компьютер
   4. Запустите приложение через терминал или любой компилятор:
      ```bash
      python MASSSSLOO.py
      ```

После запуска откроется окно приложения, где вы сможете выбрать интересующую вас категорию блюд из выпадающего списка и воспользоваться функцией переключения темы оформления.
