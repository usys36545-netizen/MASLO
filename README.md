# Приложение MASLO

## Общая информация
Это PyQt5-приложение представляет собой интерактивное руководство по приготовлению различных блюд. Приложение состоит из UI-файла (MASLO.ui) и основного Python-скрипта (MASSSSLOO.py).

## Структура проекта

### Файл MASLO.ui
- XML-файл, описывающий пользовательский интерфейс
- Содержит:
  - Комбобокс (QComboBox) с различными блюдами
  - Метку (QLabel) для отображения информации о выбранном блюде
  - Чекбокс для переключения темного режима
- Изначально содержит HTML-форматированный текст с жирным шрифтом для "Добро пожаловать!" в метке
- Добавлены комментарии для лучшего понимания структуры интерфейса

### Файл MASSSSLOO.py
- Основной Python-скрипт приложения
- Содержит класс `App`, наследуемый от `QWidget`
- Функциональность:
  - Загрузка UI из MASLO.ui
  - Обработка событий комбобокса
  - Изменение текста метки в зависимости от выбранного элемента
  - Поддержка форматирования текста (размер шрифта, жирность)
- Добавлены подробные комментарии и docstrings для лучшей читаемости кода
- Реализована логика для отображения "Добро пожаловать!" жирным шрифтом при выборе элемента "Начало" (индекс 0)

## Основные функции

### Обработка комбобокса
- При выборе элемента "Начало" (индекс 0):
  - Устанавливается HTML-форматированный текст с жирным шрифтом для "Добро пожаловать!"
  - Используется span с font-weight:600; только для слова "Добро пожаловать!"
  - Остальной текст остается обычным шрифтом
  - Размер шрифта устанавливается на 28pt
- При выборе элемента "Рис" (индекс 3):
  - Устанавливается тестовый текст
  - Размер шрифта устанавливается на 20pt

## Использование
- При запуске приложения отображается окно с комбобоксом
- При выборе различных элементов в комбобоксе изменяется текст в метке
- Для включения темного режима используется чекбокс

## Технические детали
- Используется PyQt5 для создания графического интерфейса
- Для форматирования текста используется HTML/CSS в строках
- События комбобокса обрабатываются через сигнал `activated`
- Добавлены подробные комментарии ко всем основным методам и элементам интерфейса для улучшения понимания кода
